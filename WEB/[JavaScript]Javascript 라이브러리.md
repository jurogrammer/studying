# 1. jQuery의 10년

### 유명했던 이유

* IE가 표준을 지키지 않고 브라우저를 만듦. 이러한 호환성 이슈를 jQuery가 줄여줆.
* 복잡한 DOM APIs를 쉽게 알려줌.
* 절차적인 사고의 흐름에 맞춰서 그대로 프로그래밍 가능.



# 2. 그러면 jQuery가 왜...

* 브라우저 호환성 이슈가 줄어 듦 (IE 점유율 감소)
* js Frameworks 들이 좀 더 추상화된 방식으로 DOM 접근을 도와줌
* 다른 라이브러리도 jQuery 지원.

* ECMAScript 2015부터 편리한 함수들 제공.
